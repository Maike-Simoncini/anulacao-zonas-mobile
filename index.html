Guia de Anulação de Zonas

Passo a passo para anulação de zonas e ativação

01

Acesse o Menu

Abra o app e clique no ícone de Menu no canto superior esquerdo para visualizar as opções de controle.

02

Gerenciamento de Zonas

Selecione Zonas para identificar os sensores instalados no sistema.

03

Procedimento de Bypass

Toque na zona desejada e utilize a função Anular (Bypass) para isolar o sensor necessário.

04

Validação do Status

Verificar o círculo laranja, que confirma visualmente que a zona foi isolada com sucesso.

05

Ativação do Sistema

Ativar o alarme pelo ícone do Cadeado, garantindo que o local esteja protegido.

Treinamento Concluído

Este guia possui caráter informativo e educacional para instruir pessoas na operação do sistema.

Material desenvolvido para fins de instrução e capacitação técnica.
